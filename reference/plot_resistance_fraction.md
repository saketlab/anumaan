# Resistance Fraction Stacked Bar

100 attributable to resistance vs baseline infectious burden, per
hospital.

## Usage

``` r
plot_resistance_fraction(hospital_daly, label_map = NULL)
```

## Arguments

- hospital_daly:

  Data frame from
  [`compute_hospital_daly()`](https://saketlab.github.io/anumaan/reference/compute_hospital_daly.md).

- label_map:

  Optional named character vector for center name labels.

## Value

A `ggplot` object.
