# Map Antibiotic Class to RR Drug Category

Maps WHO antibiotic classes to RR drug categories for burden estimation.

## Usage

``` r
map_class_to_rr(data, class_col = "antibiotic_class")
```

## Arguments

- data:

  Data frame

- class_col:

  Character. Antibiotic class column. Default "antibiotic_class".

## Value

Data frame with rr_drug column added
